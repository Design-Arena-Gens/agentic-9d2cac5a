# Chess Progress Reporter

An AI-powered daily progress tracking and reporting system for chess students. This application automatically collects data about student activities, puzzle solving, sessions, and leaderboard changes, then generates comprehensive daily reports.

## Features

- **Automated Data Collection**: Tracks puzzle attempts, session durations, and student progress
- **Daily Reports**: Generates detailed reports every morning at 8 AM
- **Student Analytics**: Individual breakdown by topic, accuracy, and performance
- **Leaderboard Tracking**: Monitors rank changes and point progressions
- **Book Library Tracking**: Records new additions to the chess book collection
- **PDF Export**: Download reports as PDF documents
- **Secure Authentication**: Admin-only access with password protection

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open http://localhost:3000 in your browser

4. Login with the default password: chess2024admin

## Deployment

Deploy to Vercel with the provided configuration.
